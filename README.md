Автоматизированный UI-тест Android-приложения VK Video.
Проверяет запуск приложения, открытие первого видео и факт воспроизведения ролика через анализ изменений изображения на экране.

Проект реализован как тестовое задание на мобильную автоматизацию.

Формулировка задачи: 

Стек: Appium, Java 22, Selenide, Android Studio (Android 11 / api 30), Maven.

Написать автотест на приложение VK Видео (com.vk.vkvideo) . Необходимо проверить, что видео (на ваш выбор) - воспроизводится. Для завершения автотеста, должны быть обработаны, как положительные - видео воспроизводится, так и негативные - видео не воспроизводится.
